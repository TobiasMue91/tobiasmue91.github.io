HOW IT WORKS▶

This converter uses a graph-based routing algorithm. Each file format is a node, and each converter is an edge connecting two formats. When you pick a target format, a breadth-first search finds the shortest conversion path — even if it requires chaining multiple converters together.

For example, converting HEIC → PDF chains: HEIC → PNG (via heic2any) → PNG → PDF (via jsPDF). You never need to think about intermediate steps.

Libraries are lazy-loaded from CDNs only when needed. If you only convert images, no document or audio libraries are downloaded. Native browser APIs (Canvas, Web Audio) are preferred over external libraries when available.

PRIVACY▶

Zero data transmission. Every conversion runs entirely in your browser using JavaScript. No file data is sent to any server. There are no analytics, no cookies, no tracking.

External CDN libraries (marked, turndown, js-yaml, mammoth, SheetJS, jsPDF, heic2any) are loaded on-demand for specific format support. These are code libraries only — they process your files locally and send nothing back.
